## Course-spring
This is a basic spring study project

### Dependencies
- git
- java ^v11
- gradle ^6.7

### Setup
- Run commands bellow:
    - `git clone https://github.com/kaio-giovanni/course-spring.git`
    - `cd course-spring`
    
### Run project
To run this project use the command bellow:
`gradle bootRun`

### Tests
To execute all tests, execute the command bellow:
`gradle test`
